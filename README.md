# AeroMonitor Project

## Overview
AeroMonitor is a project designed to monitor various environmental parameters. It provides real-time data collection, analysis, and visualization to help users understand and respond to changes in their environment.

## Features
- Real-time monitoring of air quality, temperature, humidity, etc.
- Data visualization through graphs and charts.
- Alerts and notifications for certain thresholds.

## Installation
To install AeroMonitor, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/joaobolado16284602-rgb/AeroMonitor.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AeroMonitor
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start monitoring, run:
```bash
npm start
```
